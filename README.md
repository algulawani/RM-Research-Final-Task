# RM-Research-Final-Task
This repo contains all the code related to the implementation of the OADN model as the proof of work for my final task. This repo also contains my grammarly report and its AI and plagiarism check results. 

As discussed earlier, there is a some confusion in the OADN paper (link: https://dl.acm.org/doi/10.1109/IJCB48548.2020.9304923), as to how many landmark points are to be selected. I have implemented for both selecting 24 and 26 landmark points.
